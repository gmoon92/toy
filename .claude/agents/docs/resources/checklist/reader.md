## 1. 대상 독자 분석 (Audience Analysis)

- [ ] **RDR-1-001: 대상 명시** - 문서의 대상 독자가 명확히 정의되어 있는가?
- [ ] **RDR-1-002: 수준 적절성** - 내용의 난이도가 대상 독자에 적합한가?
- [ ] **RDR-1-003: 선행지식** - 필요한 선행지식이 명시되어 있는가?
- [ ] **RDR-1-004: 확대 가능성** - 더 넓은 독자층을 위한 확대 방안이 있는가?

## 2. 이핏도 검증 (Comprehension)

- [ ] **RDR-2-001: 직관적 설명** - 핵심 개념이 직관적으로 설명되는가?
- [ ] **RDR-2-002: 단계별 접근** - 복잡한 과정이 단계별로 나뉘어 있는가?
- [ ] **RDR-2-003: 시각적 보조** - 다이어그램, 표, 이미지가 이해를 돕는가?
- [ ] **RDR-2-004: 충분한 맥락** - 새로운 개념에 충분한 배경 설명이 있는가?
- [ ] **RDR-2-005: 용어 정의** - 전문 용어가 적절한 시점에 정의되는가?
- [ ] **RDR-2-006: 링크 제공** - 추가 학습을 위한 참조 링크가 있는가?
- [ ] **RDR-2-007: 갑작스런 전환** - 주제 전환이 너무 급작스럽지 않은가?
- [ ] **RDR-2-008: 누락된 단계** - 중요한 중간 단계가 생략되어 있지 않은가?
- [ ] **RDR-2-009: 암묵적 가정** - 독자가 알 것이라 암묵적으로 가정하는 것은 없는가?

## 3. 실용성 평가 (Practical Value)

- [ ] **RDR-3-001: 즉시 적용** - 읽은 내용을 즉시 적용할 수 있는가?
- [ ] **RDR-3-002: 실행 가능한 조치** - 구체적인 실행 단계가 제공되는가?
- [ ] **RDR-3-003: 성과 측정** - 성공을 평가할 수 있는 기준이 있는가?
- [ ] **RDR-3-004: 시간 대비 가치** - 투자한 시간에 비해 얻는 가치가 충분한가?

## 4. 예시 관련성 (Examples)

- [ ] **RDR-4-001: 현실적 예시** - 예시가 현실적이고 관련성이 있는가?
- [ ] **RDR-4-002: 다양한 사례** - 다양한 사용 사례를 커버하는가?
- [ ] **RDR-4-003: 복잡도 단계** - 간단한 예시부터 복잡한 예시까지 단계별로 제공되는가?
- [ ] **RDR-4-004: 독자 연결** - 예시가 독자의 실제 상황과 연결되는가?

## 5. 참여도 (Engagement)

- [ ] **RDR-5-001: 훅 품질** - 서례이 독자의 관심을 끌 수 있는가?
- [ ] **RDR-5-002: 흥미 유지** - 문서 전체에 걸쳐 흥미를 유지하는가?
- [ ] **RDR-5-003: 상호작용** - 독자가 직접 시도핸 것을 권장하는가?
- [ ] **RDR-5-004: 행동 촉구** - 명확한 다음 단계/행동을 제시하는가?

## 6. 접근성 (Accessibility)

- [ ] **RDR-6-001: 소제목 활용** - 스캐닝을 위한 명확한 소제목이 있는가?
- [ ] **RDR-6-002: 요약 제공** - 각 섹션의 핵심을 요약하는가?
- [ ] **RDR-6-003: 길이 적절성** - 문서 길이가 목적에 적합한가?
- [ ] **RDR-6-004: 언어 장벽** - 비영어권 독자를 위한 고려가 있는가?
