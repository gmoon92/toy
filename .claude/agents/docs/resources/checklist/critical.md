# 비판적 검증 체크리스트 (Critical Mode)

> "This design is wrong until proven otherwise"

## 마인드셋

- "What assumptions are being made?"
- "When would this fail?"
- "Is there a simpler explanation?"
- "Are we confusing correlation with causation?"

## 1. 가정 발굴 (Assumption Hunting)

- [ ] **명시되지 않은 전제**: 작성자가 가정하고 있지만 명시하지 않은 전제는 무엇인가?
- [ ] **독자 수준 가정**: 독자가 특정 지식을 알고 있다고 가정하는가?
- [ ] **환경 가정**: 특정 환경/설정이 있다고 가정하는가?
- [ ] **사용 사례 가정**: 특정 사용 사례에 최적화되어 다른 경우를 무시하는가?

## 2. 실패 시나리오 (Failure Scenarios)

- [ ] **예외 처리**: 예외 상황이나 에러 케이스가 다뤯는가?
- [ ] **경계 조건**: 극단적인 입력값(빈 값, 매우 큰 값, 특수문자) 처리는?
- [ ] **경쟁 조건**: 동시성/타이밍 관련 문제가 있는가?
- [ ] **리소스 한계**: 메모리, 디스크, 네트워크 제한 상황에서의 동작은?
- [ ] **외부 의존성 실패**: 외부 서비스/API가 실패할 경우의 처리는?

## 3. 논리적 일관성 (Logical Consistency)

- [ ] **모순 검출**: 문서 내 서로 모순되는 주장이 있는가?
- [ ] **결론-근거**: 제시된 결론이 근거를 충분히 뒷받침하는가?
- [ ] **순환 논리**: 순환 논법이 사용되고 있지 않은가?
- [ ] **거짓 원인**: 상관관계를 인과관계로 오해하고 있지 않은가?
- [ ] **흑백 논리**: 복잡한 문제를 지나치게 단순화하고 있지 않은가?

## 4. 과장된 주장 (Exaggerated Claims)

- [ ] **검증되지 않은 성능**: "~배 빠르다" 등의 주장에 근거가 있는가?
- [ ] **절대적 표현**: "항상", "절대", "완벽" 등의 표현이 적절한가?
- [ ] **미래 예측**: 불확실한 미래를 확신에 차게 예측하는가?
- [ ] **범용성 과장**: 특정 상황에만 적용되는 해결책을 보편화하는가?

## 5. 맥락 누락 (Missing Context)

- [ ] **제약조건**: 해결책의 제약조건이 명시되어 있는가?
- [ ] **대안 검토**: 다른 대안을 고려하고 비교했는가?
- [ ] **트레이드오프**: 장점만 강조하고 단점을 숨기는가?
- [ ] **역사적 맥락**: 왜 이런 접근법을 선택했는지 배경이 있는가?

## 6. 확인 편향 (Confirmation Bias)

- [ ] **성공 사례만**: 성공 사례만 제시하고 실패 사례는 무시하는가?
- [ ] **선택적 증거**: 불리한 증거는 제외하고 유리한 것만 선택하는가?
- [ ] **반례 고려**: 반례 가능성을 고려했는가?

---

## 심각도 분류

| 등급 | 기준 | 처리 |
|-----|------|------|
| **CRITICAL** | 근본적인 가정 오류, 심각한 논리적 모순 | 반드시 수정 |
| **WARNING** | 실패 가능성, 부분적 과장 | 수정 권장 |
| **INFO** | 추가 검토 권장 | 선택적 반영 |

---

## 출력 형식

```markdown
## 비판적 검토 (critical)

### Critical Issues
- [ ] [위치]: [발견된 가정/문제]
  Impact: [이 가정이 틀릴 때의 영향]
  Suggested fix: [수정 방안]

### Warnings
- [ ] [위치]: [실패 가능성]
  Risk level: [High/Medium/Low]
  Mitigation: [완화 방안]

### Questions
- [ ] [위치]: [검증되지 않은 주장]
  Need: [필요한 검증/근거]

### Logical Inconsistencies
- [위치1] contradicts [위치2]: [모순 설명]

### Missing Context
- [주제]: [누락된 맥락] - [왜 필요한지]
```
