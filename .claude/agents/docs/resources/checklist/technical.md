# 기술 정확성 검증 체크리스트 (Technical Mode)

## 1. 코드 예제 검증 (Code Examples)

- [ ] **문법 오류 없음**: 모든 코드가 문법적으로 올바른가?
- [ ] **실행 가능**: 코드가 복사-붙여넣기로 실행 가능한가?
- [ ] **완전한 예제**: 필요한 import/include가 포함되어 있는가?
- [ ] **예상 출력**: 예상되는 출력 결과가 명시되어 있는가?
- [ ] **오류 처리**: 예외 상황에 대한 처리가 포함되어 있는가?

## 2. 용어 정의 (Terminology)

- [ ] **미정의 용어 없음**: 모든 기술 용어가 적절히 정의되는가?
- [ ] **용어 일관성**: 동일 개념에 대해 일관된 용어를 사용하는가?
- [ ] **약어 설명**: 처음 사용되는 약어가 풀어서 설명되는가?
- [ ] **번역 일관성**: 외래 용어의 한글 번역이 일관되는가?

## 3. 버전 정보 (Version Information)

- [ ] **버전 명시**: 사용된 기술/라이브러리의 버전이 명시되어 있는가?
- [ ] **최신 버전**: 참조하는 API/라이브러리가 최신 버전인가?
- [ ] **버전 호환성**: 다른 버전과의 호환성이 명시되어 있는가?
- [ ] **폐기 예정 기능**: 사용된 기능 중 폐기 예정(deprecated)인 것이 있는가?

## 4. 환경 설정 (Environment Setup)

- [ ] **의존성 명시**: 필요한 의존성(패키지, 라이브러리)이 명시되어 있는가?
- [ ] **설치 방법**: 의존성 설치 방법이 제공되는가?
- [ ] **환경 변수**: 필요한 환경 변수 설정이 설명되는가?
- [ ] **최소 요구사항**: 하드웨어/소프트웨어 최소 요구사항이 명시되어 있는가?
- [ ] **권한 설정**: 필요한 권한이나 인증 설정이 설명되는가?

## 5. 명령어 및 설정 (Commands & Configuration)

- [ ] **실행 가능한 명령어**: 제시된 명령어가 실제로 작동하는가?
- [ ] **설정 파일 예시**: 설정 파일의 완전한 예시가 제공되는가?
- [ ] **경로 처리**: 파일 경로가 적절히 처리되었는가 (절대/상대 경로)?
- [ ] **플랫폼 고려**: OS별 차이점이 언급되어 있는가?

## 6. 보안 고려사항 (Security)

- [ ] **민감 정보 노출**: 하드코딩된 비밀번호/키가 없는가?
- [ ] **SQL 인젝션**: 사용자 입력이 적절히 검증/이스케이프되는가?
- [ ] **XSS 방지**: 웹 관련 코드에서 XSS 취약점이 없는가?
- [ ] **보안 모범사례**: 보안 관련 모범사례가 따르는가?

## 7. 외부 도구 검증 (External Verification)

- [ ] **Context7 검증**: Claude API, MCP, LSP 등 관련 기술은 Context7로 최신 문서와 대조하였는가?
- [ ] **공식 문서 참조**: 공식 문서와 일치하는 정보인가?
- [ ] **버전 호환성**: Claude Code 버전과 호환되는 내용인가?
- [ ] **정확성 주석**: 예시/추정치는 "예시", "대략" 등으로 명시하였는가?

---

## 심각도 분류

| 등급 | 기준 | 처리 |
|-----|------|------|
| **CRITICAL** | 코드가 실행되지 않거나 심각한 보안 문제 | 반드시 수정 |
| **WARNING** | 버전 오래됨, 일부 기능 미작동 가능 | 수정 권장 |
| **INFO** | 추가 설명 권장 | 선택적 반영 |

## 출력 형식

```markdown
## 기술 검증 (technical)

### 코드 예제 검증 [등급]
- [x] 모든 코드 블록 문법 오류 없음
- [ ] [line X]: 문법 오류 - [문제 설명]
- [ ] [line X]: 실행 불가능한 코드 - [원인]

### 용어 검증 [등급]
- [x] 모든 기술 용어 적절히 정의됨
- [ ] 미정의 용어: [용어] - [위치]: [정의 제안]
- [ ] 불일치 용어: [용어A] vs [용어B] - [통일 제안]

### 버전 정보 [등급]
- [x] 모든 참조가 최신 버전
- [ ] 오래된 API 참조: [API] - [위치]: [최신 버전으로 업데이트 제안]

### 환경 설정 [등급]
- [x] 필요한 설정 정보 모두 포함
- [ ] 누락된 의존성: [의존성] - [위치]: [설정 예시]

### 보안 검증 [등급]
- [x] 보안 모범사례 준수
- [ ] 보안 이슈: [문제] - [위치]: [수정 제안]
```
