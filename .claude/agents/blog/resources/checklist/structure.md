## 1. 정보 구조 (Information Architecture)

- [ ] **STR-1-001: 논리적 그룹화** - 관련 내용이 적절히 그룹화되어 있는가?
- [ ] **STR-1-002: 계층 깊이** - 헤딩 레벨이 적절한 깊이(3-4단계)를 유지하는가?
- [ ] **STR-1-003: 일관된 패턴** - 유사한 내용이 일관된 구조로 제시되는가?
- [ ] **STR-1-004: 중요도 순서** - 중요한 내용이 먼저 제시되는가?

## 2. 중복 검사 (Redundancy Check)

- [ ] **STR-2-001: 섹션 간 중복** - 같은 내용이 여러 섹션에 반복되지 않는가?
- [ ] **STR-2-002: 개념 중복** - 동일 개념이 다른 표현으로 중복 설명되지 않는가?
- [ ] **STR-2-003: 예시 중복** - 동일한 예시가 불필요하게 반복되지 않는가?
- [ ] **STR-2-004: 참조 중복** - 동일한 참조가 여러 번 언급되지 않는가?

## 3. 흐름 분석 (Flow Analysis)

- [ ] **STR-3-001: 자연스러운 전환** - 섹션 간 전환이 매끄러운가?
- [ ] **STR-3-002: 인과 관계** - 원인-결과 관계가 논리적으로 연결되는가?
- [ ] **STR-3-003: 시간 순서** - 시간적 흐름이 있는 경우 순서가 올바른가?
- [ ] **STR-3-004: 난이도 순서** - 개념이 난이도 순으로 제시되는가?
- [ ] **STR-3-005: 독자 여정** - 독자가 목표 지점까지 명확한 경로를 따라가는가?

## 4. 균형 평가 (Balance Assessment)

- [ ] **STR-4-001: 섹션 길이** - 각 섹션의 길이가 적절히 균형을 이루는가?
- [ ] **STR-4-002: 중요도 비율** - 중요한 주제에 충분한 분량이 할당되었는가?
- [ ] **STR-4-003: 깊이 일관성** - 섹션 간 설명의 깊이가 일관되는가?
- [ ] **STR-4-004: 예시 분배** - 예시가 전체 문서에 고르게 분포되어 있는가?

## 5. 네비게이션 (Navigation)

- [ ] **STR-5-001: 목차 존재** - 긴 문서에 목차(Table of Contents)가 있는가?
- [ ] **STR-5-002: 앵커 링크** - 헤딩에 앵커 링크가 제공되는가?
- [ ] **STR-5-003: 상위/하위 이동** - 계층 구조 내 이동이 용이한가?
- [ ] **STR-5-004: 관련 링크** - 관련 섹션 간 연결(link)이 있는가?
- [ ] **STR-5-005: 검색 가능성** - 헤딩이 검색하기 쉬운 키워드를 포함하는가?

## 6. 헤딩 품질 (Heading Quality)

- [ ] **STR-6-001: 명확성** - 각 헤딩이 섹션 내용을 명확히 설명하는가?
- [ ] **STR-6-002: 일관된 형식** - 헤딩 형식(명사형, 동사형 등)이 일관되는가?
- [ ] **STR-6-003: 계층 일관성** - 헤딩 레벨이 논리적 계층을 따르는가?
- [ ] **STR-6-004: 길이 적절성** - 헤딩이 너무 길거나 짧지 않은가?
- [ ] **STR-6-005: SEO 고려** - 헤딩이 검색 엔진에 적합한 키워드를 포함하는가?

## 7. 시각화 검토 (Visualization)

- [ ] **STR-7-001: 적절한 표 사용** - 표는 비교/정보 전달에 효과적으로 사용되는가?
- [ ] **STR-7-002: 과도한 시각화** - 표/다이어그램이 너무 많아 서술을 방해하지 않는가?
- [ ] **STR-7-003: 서술 vs 표** - 복잡한 내용은 서술이 더 적절한가?
- [ ] **STR-7-004: 시각적 균형** - 텍스트와 시각적 요소의 균형이 적절한가?
- [ ] **STR-7-005: 간결한 표** - 표의 항목이 너무 많지 않은가? (5-7개 권장)

## 8. 개선 제안 (Improvement Suggestions)

- [ ] **STR-8-001: 구조 재구성** - 더 나은 정보 구조를 제안할 수 있는가?
- [ ] **STR-8-002: 섹션 통합** - 통합할 수 있는 중복 섹션이 있는가?
- [ ] **STR-8-003: 섹션 분리** - 분리하면 더 명확해질 섹션이 있는가?
- [ ] **STR-8-004: 순서 변경** - 순서를 바꾸면 흐름이 개선되는가?
