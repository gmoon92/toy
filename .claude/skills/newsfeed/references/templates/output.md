# 출력 템플릿 (Output Templates)

## 평가 기준

| 등급               | 기준                                                | 조치     |
|:-----------------|:--------------------------------------------------|:-------|
| [우수] **완벽함**     | 모든 필수 항목 충족 + 권장 항목 80% 이상                        | [승인]   |
| [우수] **우수함**     | 모든 필수 항목 충족                                       | [승인]   |
| [보통] **보통**      | 필수 항목 1-2개 누락                                     | [경고]   |
| [개선필요] **개선 필요** | 필수 항목 3개 이상 누락                                    | 수정 권장  |
| [개선필요] **불합격**   | `name`/`description` 누락 또는 `description` 1024자 초과 | [필수수정] |
