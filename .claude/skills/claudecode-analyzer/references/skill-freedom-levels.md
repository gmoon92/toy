# 스킬 지시 자유도 수준 (Skill Freedom Levels)

스킬 문서의 지시 강도가 task의 fragility(깨지기 쉬움)와 일치하는지 검증하는 기준.

검증 포인트: **task의 위험도에 비해 지시가 너무 느슨하거나 너무 경직되어 있지 않은가?**

---

## 자유도 스펙트럼

```
고자유도 (High)                       저자유도 (Low)
텍스트 가이드 ←——————————————→ 정확한 스크립트
```

---

## 고자유도 (High Freedom)

### 적합한 상황

- 여러 유효한 접근 방식이 존재
- 결정이 컨텍스트에 따라 달라짐
- 창의적 해결이 결과를 향상
- 각 실행 환경이 다름

### 특징

- 원칙/가이드라인 제공
- Claude가 판단해서 실행
- SHOULD/권장 표현 사용

### 예시: 코드 리뷰

```markdown
## Code Review Process
1. 코드 구조와 구성 분석
2. 잠재적 버그와 엣지 케이스 확인
3. 가독성 개선 제안
4. 프로젝트 컨벤션 준수 확인
```

**왜 고자유도?** 코드베이스마다 다름. 발견한 내용에 따라 접근 방식 조정 필요.

---

## 중간 자유도 (Medium Freedom)

### 적합한 상황

- 선호하는 패턴이 존재하지만 유연성 필요
- 설정값에 따라 동작이 달라짐
- 구조는 도움이 되지만 엄격한 강제는 불필요

### 특징

- 템플릿/패턴 제공
- 파라미터 조정 허용
- SHOULD/권장 + 명시적 옵션

### 예시: API 연동

```markdown
## API Request Pattern
이 구조를 기반으로, 상황에 맞게 timeout/retries 조정:
response = requests.post(endpoint, json=data, timeout=30, retries=3)
```

---

## 저자유도 (Low Freedom)

### 적합한 상황

- 잘못 실행하면 데이터 손상/시스템 오류 발생
- 정확한 순서가 필수
- 일관성이 크리티컬
- 모호함이 허용되지 않음

### 특징

- 정확한 스크립트 실행 지시
- MANDATORY/DO NOT 강한 지시어
- EXECUTE_SCRIPT 패턴

### 예시: DB 마이그레이션

```markdown
**MANDATORY: 이 스크립트를 정확히 실행**
python scripts/migrate.py --verify --backup

DO NOT 명령어 수정 또는 플래그 추가.
```

## 자유도 결정 매트릭스

| Task 특성 | 자유도 | 지시 유형 |
|:---------|:-------|:---------|
| 여러 유효한 해결책 존재 | 고 | 원칙, 가이드라인 |
| 컨텍스트 의존적 결정 | 고 | 휴리스틱 |
| 선호 패턴 존재, 유연성 필요 | 중간 | 파라미터화된 템플릿 |
| 깨지기 쉬운 작업 | 저 | 정확한 스크립트 |
| 순서 준수 필수 | 저 | 단계별 명령 |
| 일관성이 크리티컬 | 저 | 변형 없는 지시 |

## 경고 신호 (Warning Signs)

### 자유도가 너무 높음 (저자유도 필요한 곳에)

- 실행마다 결과가 다름
- 중요한 단계가 가끔 건너뛰어짐
- "consider using" 등 약한 표현이 위험한 작업에 사용됨

**수정**: 강한 지시어 추가, 스크립트 추출, MANDATORY 패턴 적용

### 자유도가 너무 낮음 (고자유도 필요한 곳에)

- 간단한 작업에 과도한 지시
- 엣지 케이스에서 경직됨
- 불필요한 스크립트 의존

**수정**: 자유도 높이기, 원칙 중심으로 재작성

---

## 검증 포인트

**스킬 검증 시 확인:**

- [ ] (WARNING) 데이터 변경/삭제/시스템 작업을 다루는 섹션에 강한 지시어(`MANDATORY`, `DO NOT`) 없음
- [ ] (WARNING) 정확한 순서가 필요한 작업에 "consider", "you might want" 등 약한 표현 사용
- [ ] (INFO) 창의적/분석 작업에 과도한 스크립트 의존 (저자유도 남용)
- [ ] (INFO) 스킬이 지원하는 모델 범위에 비해 자유도 수준이 부적절함
