## Quick Reference

### Three Foundation Principles

1. **Be Explicit** - Clear, specific instructions
2. **Add Context** - Explain why, not just what
3. **Show Examples** - Complete, realistic demonstrations

### For Long-Running Tasks

- Phase-based reference loading
- Metadata caching for state
- Structured formats (JSON) for data
- Text files for progress notes
- Git for history tracking

### Writing Style

- Concise and direct
- Active voice, present tense
- Imperative for instructions
- Prose for explanations
- Bullets for lists only
