# 서브 에이전트에서 사용 가능한 도구

서브 에이전트는 Claude Code의 내부 도구에 자유롭게 접근할 수 있습니다. `tools` 필드를 생략하면 메인 스레드의 모든 도구를 상속합니다.

## 핵심 도구

| 도구 | 설명 |
|------|------|
| `Read` | 파일 내용 읽기 |
| `Write` | 파일 생성 또는 덮어쓰기 |
| `Edit` | 기존 파일 정밀 수정 |
| `Glob` | 패턴 매칭으로 파일 검색 |
| `Grep` | 정규식으로 파일 내용 검색 |
| `Bash` | 셸 명령 실행 |
| `Task` | 서브 에이전트 생성 (서브 에이전트 내에서는 비권장) |

## 상호작용 도구

| 도구 | 설명 |
|------|------|
| `AskUserQuestion` | 사용자에게 명확화 질문 |

## 웹 도구

| 도구 | 설명 |
|------|------|
| `WebFetch` | 웹 콘텐츠 가져오기 및 처리 |
| `WebSearch` | 웹 검색 |

## IDE 도구 (사용 가능한 경우)

| 도구 | 설명 |
|------|------|
| `mcp__ide__getDiagnostics` | VS Code에서 언어 진단 정보 가져오기 |
| `mcp__ide__executeCode` | Jupyter 커널에서 코드 실행 |

## MCP 도구

서브 에이전트는 설정된 MCP 서버의 도구에도 접근할 수 있습니다. MCP 도구명은 `mcp__<서버>__<도구>` 형식을 따릅니다.

## 자주 사용하는 도구 조합

### 읽기 전용 조사
```
tools: Read, Grep, Glob, Bash
```
용도: 코드 분석, 문서 검토, 코드베이스 탐색

### 코드 수정
```
tools: Read, Write, Edit, Grep, Glob, Bash
```
용도: 기능 구현, 버그 수정, 리팩터링

### 최소 쓰기 권한
```
tools: Read, Grep, Glob
```
용도: 보안 감사, 코드 리뷰 (보고 전용)

### 전체 접근
`tools` 필드를 생략하면 사용 가능한 모든 도구를 상속합니다.
