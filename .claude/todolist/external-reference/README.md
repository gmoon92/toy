# External Reference

다른 프로젝트에서 참조용으로 수집한 Claude Code 메타 문서들.

## 목적

이 디렉토리의 문서들은 **참고용**으로, 외부 프로젝트의 구현 방식을 분석하고 아이디어를 얻기 위해 수집되었습니다.

## 구조

```
external-reference/
├── agents/          # 외부 프로젝트의 에이전트 정의
├── commands/blog/   # 블로그 워크플로우 커맨드
└── skills/          # 외부 프로젝트의 스킬 구현
```

## 현재 프로젝트와의 관계

- `.claude/agents/docs/` - 현재 프로젝트의 활성 에이전트
- `.claude/skills/` - 현재 프로젝트의 활성 스킬
- `.claude/docs/external-reference/` - **참조용 외부 자료** (이 디렉토리)
