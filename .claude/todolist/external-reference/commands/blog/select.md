## 인사이트 선정

추출된 인사이트 중 블로그 글로 작성할 주제를 선택합니다.

### 추출된 인사이트

@content/insights/*.md

### 선정 프로세스

1. 인사이트 목록을 사용자에게 제시합니다
2. 각 인사이트에 대해 설명합니다:
   - 제목과 핵심 아이디어
   - 점수와 추천 이유
   - 예상 난이도
3. 사용자가 번호 또는 제목으로 선택하도록 합니다
4. 선택된 인사이트를 `content/selected/{date}-{topic-slug}.md`에 저장합니다
5. 워크플로우 상태를 업데이트합니다

### 선택 방식

사용자에게 다음과 같이 질문합니다:
- "위 인사이트 중 블로그 글로 작성하고 싶은 주제를 선택해주세요 (번호 또는 제목)"
- 여러 개 선택 가능 (쉼표로 구분)
