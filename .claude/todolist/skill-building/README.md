# Skill Building

Claude Code Skill 개발 방법론 및 가이드.

## 목적

일관되고 효과적인 Skill을 개발하기 위한 원칙, 패턴, 템플릿을 제공합니다.

## 구조

```
skill-building/
├── SKILL.md                    # 스킬 빌더 메인 문서
├── prompting/                  # 프롬프트 작성 가이드
│   ├── principles/            # 프롬프트 원칙
│   ├── guidelines/            # 구체적 가이드라인
│   └── reference/             # 참조 자료
├── skills/                     # 스킬 개발 가이드
│   ├── principles/            # 스킬 설계 원칙
│   ├── guidelines/            # 구현 가이드라인
│   └── reference/             # 체크리스트, 퀵 가이드
├── workflows/                  # 스킬 생성/수정 워크플로우
└── templates/                  # 스킬 템플릿
```

## 사용법

1. **새 스킬 개발**: `workflows/creating-skills.md` 참조
2. **기존 스킬 수정**: `workflows/modifying-skills.md` 참조
3. **프롬프트 작성**: `prompting/principles/` 및 `guidelines/` 참조
4. **스킬 템플릿**: `templates/skill-template.md` 사용
