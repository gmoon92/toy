# [프롬프트 템플릿 초안 자동 생성](https://platform.claude.com/docs/en/build-with-claude/prompt-engineering/prompt-generator)

---

> **참고**
>
> 프롬프트 생성기는 확장된 사고(extended thinking) 기능을 포함한 모든 Claude 모델과 호환됩니다. 확장된 사고 모델에 특화된 프롬프팅 팁은 [여기](../02-capabilities/03-extended-thinking.md)를 참조하세요.

때로는 AI 모델을 사용할 때 가장 어려운 부분이 효과적으로 프롬프트를 작성하는 방법을 찾는 것입니다. 이를 돕기 위해 저희는 Claude가 특정 작업에 맞춤화된 고품질 프롬프트 템플릿을 생성하도록 안내하는 프롬프트 생성 도구를 만들었습니다. 이러한 템플릿은 프롬프트 엔지니어링 모범 사례를 많이 따릅니다.

프롬프트 생성기는 "빈 페이지 문제"를 해결하는 도구로서 특히 유용하며, 추가 테스트와 반복 작업을 위한 출발점을 제공합니다.

> **팁**
>
> [Console](/dashboard)에서 프롬프트 생성기를 직접 사용해보세요.

기본 프롬프트와 아키텍처를 분석하는 데 관심이 있다면 [프롬프트 생성기 Google Colab 노트북](https://anthropic.com/metaprompt-notebook/)을 확인해보세요. 여기서 코드를 쉽게 실행하여 Claude가 대신 프롬프트를 구성하도록 할 수 있습니다.

> **참고**
>
> Colab 노트북을 실행하려면 [API 키](/settings/keys)가 필요합니다.
