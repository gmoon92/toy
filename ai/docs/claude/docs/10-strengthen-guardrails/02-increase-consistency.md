> **보장된 JSON 스키마 준수가 필요한 경우**
>
> Claude가 특정 스키마에 부합하는 유효한 JSON을 항상 출력하도록 해야 한다면, 아래의 프롬프트 엔지니어링 기법
> 대신 [구조화된 출력](../02-capabilities/15-structured-outputs.md)을 사용하세요. 구조화된 출력은 보장된 스키마 준수를 제공하며 이러한 사용 사례를 위해 특별히 설계되었습니다.
>
> 아래의 기법들은 일반적인 출력 일관성이나 엄격한 JSON 스키마를 넘어선 유연성이 필요한 경우에 유용합니다.


Claude의 응답을 더욱 일관되게 만드는 방법은 다음과 같습니다:

## 원하는 출력 형식 지정

JSON, XML 또는 사용자 정의 템플릿을 사용하여 원하는 출력 형식을 정확하게 정의하면, Claude가 필요한 모든 출력 형식 요소를 이해할 수 있습니다.

<details>

| 역할        | 내용                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
    |-----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| User      | 당신은 고객 인사이트 AI입니다. 이 피드백을 분석하고 다음 키를 가진 JSON 형식으로 출력하세요: "sentiment" (positive/negative/neutral), "key_issues" (목록), "action_items" (각각 "team"과 "task"를 가진 딕셔너리 목록).<br/><br/>"저는 3년간 충성스러운 사용자였지만, 최근 UI 업데이트는 재앙입니다. 기본 기능을 찾는 것이 이제 보물찾기가 되어버렸어요. 게다가, 새로운 '프리미엄' 가격은 터무니없습니다. 이것이 즉시 수정되지 않으면 다른 서비스로 전환을 고려하고 있습니다."                                                                                                                                                                                                                                                                  |

## 맥락적 일관성을 위한 검색 사용

일관된 맥락이 필요한 작업(예: 챗봇, 지식 베이스)의 경우, 검색을 사용하여 Claude의 응답을 고정된 정보 세트에 기반하도록 합니다.

<details>
<summary>예시: IT 지원 일관성 향상</summary>

| 역할        | 내용                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
    |-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| User      | 당신은 지식 베이스 데이터를 활용하는 IT 지원 AI입니다. 다음은 지식 베이스의 항목들입니다:<br/><br/>\<kb><br/>  \<entry><br/>    \<id>1\</id><br/>    \<title>Active Directory 비밀번호 재설정\</title><br/>    \<content>1. password.ourcompany.com으로 이동<br/>2. 사용자 이름 입력<br/>3. "비밀번호 분실" 클릭<br/>4. 이메일 지침 따르기\</content><br/>  \</entry><br/>  \<entry><br/>    \<id>2\</id><br/>    \<title>VPN 연결\</title><br/>    \<content>1. 소프트웨어 센터에서 GlobalProtect 설치<br/>2. ## 복잡한 작업을 위한 프롬프트 체이닝

복잡한 작업을 더 작고 일관된 하위 작업으로 나눕니다. 각 하위 작업은 Claude의 전체 주의를 받게 되어, 확장된 워크플로우에서 비일관성 오류를 줄입니다.
