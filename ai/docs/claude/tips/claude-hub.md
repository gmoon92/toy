# Claude code hub

- https://github.com/jarrodwatts/claude-hud/tree/main

## 1. 플러그인 설치

마켓플레이스를 추가하고 플러그인을 추가합니다.

```bash
claude plugin marketplace add jarrodwatts/claude-hud \
  & claude plugin install claude-hud
```

## 2. Setup

클로드 코드 CLI 진입 후 `/claude-hud:setup` 설정을 적용하면 즉시 하단에 HUD가 나타납니다.  

```bash
/claude-hud:setup
```
